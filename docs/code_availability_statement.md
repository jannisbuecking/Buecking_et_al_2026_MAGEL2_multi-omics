# Draft Code Availability Statement

RNA-seq and mass-spectrometry data have been deposited at GEO and will be publicly available as of the date of publication. Accession numbers will be listed in the key resources table. The MAMOTH Web Portal can be accessed at https://jannisbuecking-mamoth.share.connect.posit.cloud/.

All custom code used for the multi-omics analysis will be publicly available at https://github.com/jannisbuecking/Buecking_et_al_2026_MAGEL2_multi-omics and archived at https://doi.org/10.5281/zenodo.19799678 as of the date of publication. The MAMMOTH Shiny web-app source code and packaged app data will be publicly available at https://github.com/jannisbuecking/MAMMOTH and archived at https://doi.org/10.5281/zenodo.19799680 as of the date of publication. Any additional information required to re-analyze the data reported in this paper is available from the lead contact upon request.
